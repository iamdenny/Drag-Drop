jindo.$A

$A() 객체는 배열(Array)을 좀 더 편리하게 다룰 수 있도록 메서드를 제공한다.

jindo.$Agent

$Agent() 객체는 운영체제, 브라우저를 비롯한 사용자 시스템 정보를 제공한다.

jindo.$Ajax

$Ajax() 객체는 다양한 개발 환경에서 Ajax 요청과 응답을 쉽게 구현하기 위한 메서드를 제공한다.

jindo.$Ajax.FrameRequest

Ajax 요청 타입이 iframe인 요청 객체를 생성하며, $Ajax() 객체에서 Ajax 요청 객체를 생성할 때 사용한다.

jindo.$Ajax.JSONPRequest

Ajax 요청 타입이 jsonp인 요청 객체를 생성하며, $Ajax() 객체에서 Ajax 요청 객체를 생성할 때 사용한다.

jindo.$Ajax.Queue

Ajax 요청을 큐에 담아 큐에 들어온 순서대로 요청을 처리한다.

jindo.$Ajax.RequestBase

Ajax 요청 객체의 기본 객체이다.

jindo.$Ajax.Response

$Ajax.Response 객체를 생성한다.

jindo.$Ajax.SWFRequest

Ajax 요청 타입이 flash인 요청 객체를 생성하며, $Ajax() 객체에서 Ajax 요청 객체를 생성할 때 사용한다.

jindo.$Class

$Class() 객체는 Jindo 프레임워크를 사용하여 객체 지향 프로그래밍 방식으로 애플리케이션을 구현할 수 있도록 지원한다.

jindo.$Cookie

$Cookie() 객체는 쿠키(Cookie)에 정보를 추가, 수정, 혹은 삭제하거나 쿠키의 값을 가져온다.

jindo.$Date

$Date() 객체는 Date 객체를 래핑(Wrapping)하여 날짜 및 시간을 처리하기 위한 확장 기능을 제공한다.

jindo.$Document

$Document() 객체는 문서와 관련된 정보를 제공한다.

jindo.$Element

$Element() 객체는 HTML 요소를 래핑(wrapping)하며, 해당 요소를 좀 더 쉽게 다룰 수 있는 기능을 제공한다.

jindo.$ElementList

$ElementList() 객체는 여러 개의 DOM 요소를 한 번에 다룰 수 있는 기능을 제공한다.

jindo.$Event

$Event() 객체는 Event 객체를 래핑하여 이벤트 처리와 관련된 확장 기능을 제공한다.

jindo.$Fn

$Fn() 객체는 Function 객체를 래핑(wrapping)하여 함수와 관련된 확장 기능을 제공한다.

jindo.$Form

$Form() 객체는 <form> 요소와 자식 요소를 제어하는 기능을 제공한다.

jindo.$H

$H() 객체는 키(key)와 값(value)을 원소로 가지는 열거형 배열인 해시(Hash)를 구현하고, 해시를 다루기 위한 여러 가지 위한 메서드를 제공한다.

jindo.$Jindo

$Jindo 객체는 프레임워크에 대한 정보와 유틸리티 함수를 제공한다.

jindo.$Json

$Json 객체는 JSON(JavaScript Object Notation)을 다루기 위한 다양한 기능을 제공한다.

jindo.$S

$S() 객체는 String 객체를 래핑(wrapping)하여 문자열을 처리하기 위한 확장 기능을 제공한다.

jindo.$Template

$Template() 객체를 사용하면 템플릿을 해석하여 템플릿에 동적으로 데이터를 적용할 수 있다.

jindo.$Window

$Window() 객체는 브라우저가 제공하는 window 객체를 래핑하고, 이를 다루기 위한 여러가지 메서드를 제공한다.
