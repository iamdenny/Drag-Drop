Class jindo.$Ajax.SWFRequest


Extends $Ajax.RequestBase.
Ajax 요청 타입이 flash인 요청 객체를 생성하며, $Ajax() 객체에서 Ajax 요청 객체를 생성할 때 사용한다.
Defined in: jindo.all.ns.js.

Method Detail
<static> jindo.$Ajax.SWFRequest.write(sSWFPath)
write() 메서드는 플래시 객체를 초기화하는 메서드로서 write() 메서드를 호출하면 통신을 위한 플래시 객체를 문서 내에 추가한다. Ajax 요청 타입이 flash이면 플래시 객체를 통해 통신한다. 따라서 $Ajax() 객체의 request 메서드가 호출되기 전에 write() 메서드를 반드시 한 번 실행해야 하며, 요소에 작성되어야 한다. 두 번 이상 실행해도 문제가 발생한다.
Parameters:
{String} sSWFPath Optional
Ajax 통신에 사용할 플래시 파일. 기본 값은 "./ajax.swf" 이다.
