{Object} clientSize()
clientSize() 메서드는 문서에서 스크롤바가 생겨 보이지 않는 부분을 제외한 영역(화면에 보이는 부분)의 가로 크기와 세로 크기 정보를 담은 객체를 반환한다. 다음은 화면에 보이는 영역의 가로, 세로 크기 정보를 담고 있는 객체의 속성을 설명한 표이다.
화면 크기 정보 객체
이름 속성 설명
width number 화면에 보이는 부분의 가로 크기. 단위는 픽셀(px)이다.
height number 화면에 보이는 부분의 세로 크기. 단위는 픽셀(px)이다. 
var size = $Document(document).clientSize();
alert('가로 : ' + size.width + ' / 세로 : ' + size.height);
Returns:
{Object} 화면에 보이는 부분의 크기를 저장하고 있는 객체.

{Element} query(sSelector)
문서를 대상으로 지정한 선택자를 만족시키는 첫 번째 요소를 반환한다. 만족하는 요소가 존재하지 않으면 null을 반환한다. $$.getSingle 함수에서 elBaseElement 파라미터에 문서를 지정한 것과 같은 의미이다. 
 $Document().query("body");  //  요소를 선택한다.
Parameters:
{String} sSelector
CSS 선택자(CSS Selector). CSS 선택자로 사용할 수 있는 패턴은 표준 패턴과 비표준 패턴이 있다. 표준 패턴은 CSS Level3 명세서에 있는 패턴을 지원한다. 선택자의 패턴에 대한 설명은 $$() 함수와 See Also 항목을 참고한다.
Returns:
{Element} 선택자 조건을 만족하는 첫 번째 요소.
See:

렌더링 방식
렌더링 모드 설명
Standards 표준 렌더링 모드.
Almost 유사 표준 렌더링 모드. 인터넷 익스플로러 외의 브라우저에서 DTD를 올바르게 지정하지 않았을 때 반환된다.
Quirks 비표준 렌더링 모드. 
var mode = $Document().renderingMode();

{Object} scrollPosition()
scrollPosition() 메서드는 현재 문서에서 스크롤바의 위치를 구한다. 다음은 스크롤바의 위치 정보를 담고 있는 객체의 속성을 설명한 표이다.
스크롤바 위치 정보 객체의 속성
이름 타입 설명
left number 수평 스크롤바의 가로 위치. 단위는 픽셀(px)이다.
height number 수직 스크롤바의 세로 위치. 단위는 픽셀(px)이다. 
var size = $Document().scrollPosition();
alert('가로 : ' + size.left + ' / 세로 : ' + size.top);
Since:
1.3.5버전부터 사용 가능
Returns:
{Object} 가로 위치는 left, 세로위치는 top 라는 키값으로 리턴된다.

{Array} xpathAll(sXPath)
문서를 대상으로 XPath 문법을 만족하는 요소를 가져온다. 지원하는 문법이 제한 적이므로 특수한 경우에만 사용할 것을 권장한다. $$.xpath() 함수에서 elBaseElement 파라미터에 문서를 지정한 것과 같은 의미이다. 
var oDocument = $Document();
alert (oDocument.xpathAll("body/div/div").length);
