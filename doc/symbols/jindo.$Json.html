Class jindo.$Json

$Json 객체는 JSON(JavaScript Object Notation)을 다루기 위한 다양한 기능을 제공한다. 생성자에 파라미터로 객체나 문자열을 입력한다. XML 형태의 문자열로 $Json() 객체를 생성하려면 fromXML() 메서드를 사용한다.
Defined in: jindo.all.ns.js.

{Boolean} compare(oData)
compare() 메서드는 Json 객체끼리 값이 같은지 비교한다. 
$Json({foo:1, bar: 31}).compare({foo:1, bar: 31});

// 결과 :
// true

$Json({foo:1, bar: 31}).compare({foo:1, bar: 1});

// 결과 :
// false

<static> {$Json} jindo.$Json.fromXML(sXML)
fromXML() 메서드는 XML 형태의 문자열을 $Json() 객체로 인코딩한다. XML 형식의 문자열에 XML 요소가 속성을 포함하고 있을 경우 해당 요소의 정보에 해당하는 내용을 하위 객체로 표현한다. 이때 요소가 CDATA 값을 가질 경우 $cdata 속성으로 값을 저장한다. 
var j1 = $Json.fromXML('only string');

// 결과 :
// {"data":"only string"}

var j2 = $Json.fromXML('Faqh%$string value');

// 결과:
// {"data":{"id":"Faqh%$","str":{"attr":"123","$cdata":"string value"}}}
Parameters:
{String} sXML
XML 형태의 문자열.
Returns:
{$Json} $Json() 객체.
{Array} get(sPath)
get() 메서드는 특정 경로(path)에 해당하는 $Json() 객체의 값을 반환한다. 
var j = $Json.fromXML('Faqh%$string value');
var r = j.get ("/data/id");

// [Faqh%$]

toObject() 메서드는 $Json() 객체를 원래의 데이터 객체로 반환한다.

// 결과 :
